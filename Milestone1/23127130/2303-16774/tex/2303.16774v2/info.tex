\section*{Data and code availability}

Sources of the data used in the paper are described in Methods.
The code and instructions for replicating the analyses, including a packaged Python code implementing
all SWB methods in a user-friendly manner, is available at Github (\url{https://github.com/sztal/swb}).
The repository provides also the datasets used in the analyses.
