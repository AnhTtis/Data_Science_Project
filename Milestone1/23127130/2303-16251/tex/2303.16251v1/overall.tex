\section{Main Result}\label{sec:overall}
We can now state and prove our main result, which ties together the results of the previous two
sections.
\begin{theorem}\label{th:overall}
For a target function $f:\R^n\to\R$ that is continuous on a bounded set $B\subset\R^n$, let there
be a base approximation $\fbN$ of the form \eqref{singleHidden} with an arbitrary number $N\geq1$
of terms which satisfies $\|\fch-\fbN\|_B\leq\epsapprox$ for some $\epsapprox>0$, with $\fch$
being some \textit{affine shift} of $f$, and with a coefficient
bound $\sum_{i=1}^N|c_i|\leq K_N$ for some $K_N>0$.\\\\
Then for any parameter set $\Theta$, mollification $\lambda>0$, and activation function $\sigma$
satisfying Assumption~\ref{sigAsmpt}, the $\normB{\cdot}$ norm of the difference between $\fch$
%between the
%mollified approximation $\fhlN$, defined in \eqref{molliFunc} for the base approximation $\fbN$,
and the random approximation $\ftr(x,\Thetabr)$,
%of the form \eqref{linCombBasis},
as defined in
Section~\ref{sec:random}, can be bounded as follows. With probability greater than 1-$\nu$, for
any $\nu\in(0,1)$, over the iid draw $\Thetabr\simi\Pl$, there is a random approximation
$\ftr(x,\Thetabr)$ satisfying
\begin{align*}
&\normB{\fch(x)-\ftr(x,\Thetabr)} \\ \nonumber
&\leq \ \epsapprox \ + \  K_N\,\Lc\, \bigg[\frac{\ZB \sqrt{n+1}}{\lambda}\, +
\frac{1}{\pmin \sqrt{R}}\left(\frac{\eta\lambda}{e}\right)^{n+1} \\ \nonumber
&\hspace{20pt} \bigg(\norm{\thmax}\ZB + \frac{|\sigma(0)|}{\Lc} 
+ \Dt\ZB\sqrt{\tfrac{1}{2}\log\left(\tfrac{1}{\nu}\right)}\,\bigg)\bigg]
\end{align*}
with $|c_j|\leq\frac{K_N}{\pmin\,R}\left(\frac{\eta\lambda}{e}\right)^{n+1}$ for all $j\in[R]$.
\end{theorem}
\begin{proof}
Applying the triangle inequality, we have 
\begin{align*}
&\normB{\fch(x)-\ftr(x,\Thetabr)} \leq \\
&\hspace{20pt} \normB{\fch-\fbN} + \normB{\fbN-\fhlN} + \normB{\fhlN-\ftr}
\end{align*}
where, by assumption, the first term is bounded by $\epsapprox$, then the second term is bounded
by Theorem~\ref{th:moll}, and the third term is bounded by Theorem~\ref{thRand}. Then we use the
$\sum_{i=1}^N|c_i|\leq K_N$ assumption to complete the proof.
\end{proof}
