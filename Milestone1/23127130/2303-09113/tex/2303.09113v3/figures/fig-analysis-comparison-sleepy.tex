        %

        %

        %
        %
        %
        %

        pivots' probabilistic vs.\ combinatorial qualities
        into \emph{\sltpps} vs.\ \emph{\sltcps}.

        but of \emph{many} consecutive \sltpps \emph{one} is a \sltcp (new combinatorial argument),
        and \sltpps are abundant (new probabilistic argument).%
