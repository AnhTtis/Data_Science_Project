\begin{abstract}
Structural chirality induces characteristic responses that change sign
with the handedness of the crystal structure. One example is
electrical magnetochiral anisotropy (eMChA), a change in resistivity
that depends linearly on the applied current and magnetic field.  A
strong eMChA was recently reported for $p$-doped tellurium.
% ($p$-Te).
% by two experimental groups. Motivated by these works,
With this motivation, we carry out an \textit{ab initio} study of its
bulk eMChA response
% of eMChA in $p$-Te
as a function of temperature and doping
concentration.
% 
% developed an \textit{ab initio} methodology
% for evaluating the eMChA response of \tents{\st{chiral}
%   noncentrosymmetric bulk} crystals.
We use the semiclassical
Boltzmann transport formalism within the constant relaxation-time
approximation to express the 
current density at order $E^2B$ in terms of
the energy dispersion, intrinsic % \st{orbital}\tents{ and spin}
magnetic moment, and Berry curvature of conduction electrons.
% which are  evaluated  using a
% Wannier-interpolation scheme.
%
% The
% resulting expression is evaluated
% % numerically
% for $p$-doped tellurium, using a Wannier-interpolation scheme to
% perform the needed Brillouin-zone integrals.
% We report the eMChA
% coefficient as a function of temperature and doping concentration, and
% compare with the available experimental data.
We find that the orbital Zeeman coupling dominates the response, with
smaller contributions coming from the spin-Zeeman coupling and from
the Berry curvature. The agreement with the available experimental
data is satisfactory.
% while Berry curvature and spin add a significantly smaller
% correction.  The developed methodology is made available via the
% open-source code WannierBerri.
\end{abstract}
