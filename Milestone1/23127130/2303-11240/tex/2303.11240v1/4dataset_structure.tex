Our \textit{TruthSocial} dataset is available for download in tab separated value (TSV) format, that utilizes a tab delimiter. Included in the dataset are the following tables:
\begin{itemize}
\item users.tsv
\item follows.tsv
\item truths.tsv
\item quotes.tsv
\item replies.tsv
\item media.tsv
\item hashtags.tsv
\item external\_urls.tsv
\item truth\_hashtag\_edges.tsv
\item truth\_media\_edges.tsv
\item truth\_external\_url\_edges.tsv
\item truth\_user\_tag\_edges.tsv
\end{itemize}

Driven by an underlying user network structure, there are a number of relations between the different tables. Relationships between users (following relationships) are enumerated via the follows table, with each entry referencing the following and the followed user via a foreign key to user entries. Truths are linked to users as well via a foreign key. Finally the rest of the tables (quotes, replies, media, hashtags, external\_urls, and their respective edge tables) all work to further contextualize the content of Truths.

As an example, we feature a relationship diagram between users, follows, and truths. A comprehensive mapping of the each table's fields and their relationships is featured with the release of our data. 

\paragraph{FAIR Principles}
We highlight here that our dataset conforms to all of the FAIR\footnote{https://force11.org/info/the-fair-data-principles/} principles.
 Our dataset is \textit{findable}, as we release it with the Zenodo data service and give it a permanent digital object identifier (DOI) (https://doi.org/10.5281/zenodo.7522646). The dataset is accessible as it is freely available on the internet and can be accessed by anyone with an internet connection. All of the data is provided as tab serpated value (TSV) files, a standard format for handling data. The metadata we collected is also included with the dataset making it \textit{accessible}. As the data is released in TSV format it is easy to process in a large variety of programming languages. The data is also easily loaded into a SQL database of a users choice and interfaced with. Thus, our dataset is \textit{interoperable}. The dataset is released publicly and along with a ReadME file that describes dataset statistics, and key relationships between the various tables. This makes our dataset \textit{reusable} by the research community.
