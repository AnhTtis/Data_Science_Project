
The contributions of this paper are:
\begin{itemize}
    \item An infinite-dimensional \ac{SOCP} that upper-bounds the chance-peak program using \ac{VAR} inequalities \cite{cantelli1929sui, vysochanskij1980justification}
    \item A convergent set of \acp{SDP} using the Moment-\ac{SOS} hierarchy to the \ac{SOCP} upper bound
    \item A verification of this approach on example polynomial \ac{SDE} systems
\end{itemize}
