We would like to thank the reviewers for their valuable comments.
We have made various corrections and clarifications throughout the paper
to improve our work. All edits are highlighted blue in the revised version of 
the paper. 

Our core effort was on addressing the following issues:
\begin{itemize}
\item Clarifying the novelty of our work;
\item Expanding the evaluation with the inclusion of a new deep learning model architecture and a new application domain;
\item  Explaining the relationship of Mobiprox and other mobile deep learning compression approaches, libraries, and code generation tools;
\item  Discussing the applicability of Mobiprox in different domains, including computer vision;
\item Clarifying Mobiprox's readiness for use with commodity mobile phones.
\end{itemize}
